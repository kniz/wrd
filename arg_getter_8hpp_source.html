3 // argument getter:
4 // this macro can pick designated indexing argument among varidics.
5 //
6 // VARG2?:
7 // NAMU_GET_VARG2_... are created for assuming varidic arguments
8 // are paired. for instance,
9 //
10 // MyMacro(int, value1, float, value2, char, value3, double, value4...)
11 //
12 // As you can see, even the macro can't distinguish whether those macro
13 // arguments are a set of two, macro developer's want to.
14 // Might can be used in this situation.
15 //
16 // usage:
17 // VARG2 macro assumes that your macro's arguments were paired with
18 // two arguments as one. like,
19 //
20 // MyMacro( int, value1, float, value2, char, value3...
21 // X_1 Y_1 X_2 Y_2 X_3 Y_3
22 // so, at example above, NAMU_GET_VARG2_X_2 will be substitued
23 // to 'float'.
24 //
25 // argument getter defines:
26 // Yeah, I know. This is looks somewhat stupid.
27 // if you have better idea, please let me know. I beg you.
