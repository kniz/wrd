8 // you can check the file recorded logs at your current work directory

10 //
11 // if you dbg version macros, _DI, _DE, _DW, then log msg will be left
12 // only if you use dbg configuration.
13 //
14 // usage:
15 // NAMU_I("just message.")
16 // NAMU_I("with format=%x string=%s", &format, format.c_str())
